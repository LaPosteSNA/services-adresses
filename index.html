Stories clients

STORY : USE CASE POUR LES CALL CENTER INSIGHT :

« Je travaille avec un call center, et je veux optimiser le parcours de mes clients et fiabiliser le recueil de leur adresse. J’ai besoin d’un outil d’aide à la saisie, paramétrable, simple d’utilisation pour des téléconseillers. La solution doit pouvoir se moduler en fonction de la recherche sur le code postal ou la commune, pour filtrer les résultats le plus finement possible ». SOLUTION : Formulaire inversé

STORY : USE CASE POUR LES SERVICES EN LIGNE INSIGHT :

« J’ai un site de ventes de services en ligne, je veux optimiser le parcours de mes clients et fiabiliser le recueil de leur adresse. J’ai besoin d’intégrer sur mon site, un module de saisie d’adresse de type formulaire. La solution doit pouvoir hiérarchiser les données, suggérer des alternatives en cas d’erreur et proposer une adresse fiable». SOLUTION : formulaire 6 lignes

STORY : USE CASE POUR LE E-COMMERCE INSIGHT :

« Je suis un site e-commerçant, je veux optimiser le parcours de mes clients et fiabiliser le recueil de leur adresse. J’ai besoin d’intégrer sur mon site, un module de saisie d’adresse sur une ligne, en mode fulltext. La solution doit pouvoir hiérarchiser les données, suggérer des alternatives en cas d’erreur et proposer une adresse fiable ». SOLUTION : Fulltext
